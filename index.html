Software කියන්නේ මොකක්ද?

පරිගණකයක හෝ ජංගම දුරකථනයක අපට අතින් ඇල්ලිය හැකි දේවල් "Hardware" ලෙස හඳුන්වද්දී, ඒ දේවල් වලට පණ දෙන, අතින් ඇල්ලිය නොහැකි වැඩසටහන් "Software" නොහොත් මෘදුකාංග ලෙස හැඳින්වේ. හරියට අපේ ශරීරය Hardware නම්, අපේ දැනුම සහ සිතිවිලි Software වගේ. Hardware කොච්චර බලගතු වුනත්, හොඳ Software නැත්නම් වැඩක් නෑ.

සම්පූර්ණ විස්තරය කියවන්න (Read More)

Software ප්‍රධාන වර්ග 3ක් ඇත:

  • System Software: මේවා තමයි පරිගණකයේ අඩිතාලම. Hardware පාලනය කරන්නේ මේවායින්. උදාහරණ: Windows, macOS, Android, Linux. මේවා නැතුව පරිගණකයක් On වෙන්නේ නෑ.
  • Application Software: අපි අපේ එදිනෙදා වැඩ කරගන්න දාගන්න මෘදුකාංග. උදාහරණ: MS Word, Photoshop, Google Chrome, Games. මේවා නැති වුනාට පරිගණකය වැඩ කරනවා, හැබැයි අපේ වැඩ කරගන්න බෑ.
  • Programming Software: අනිත් මෘදුකාංග හදන්න උදව් වෙන මෘදුකාංග. උදාහරණ: VS Code, Python Compilers.

කොහොමද Software හැදෙන්නේ?

මෘදුකාංග ඉංජිනේරුවන් (Software Engineers) විවිධ පරිගණක භාෂා (Coding Languages) පාවිච්චි කරලා තමයි මේවා ලියන්නේ. Java, C++, Python, JavaScript වගේ භාෂා දහස් ගණනක් ලෝකයේ තියෙනවා.

Web Site එකක් හැදෙන්නේ කොහොමද?

Website (වෙබ් අඩවියක්) කියන්නේ අන්තර්ජාලය හරහා ලෝකයේ ඕනෑම කෙනෙක්ට පිවිසිය හැකි, එකිනෙකට සම්බන්ධ පිටු (Web Pages) සමූහයක්. මේක හරියට අන්තර්ජාලයේ තියෙන ඩිජිටල් පොතක් වගේ. ව්‍යාපාරයකට, පාසලකට හෝ පුද්ගලයෙක්ට තමන්ගේම අනන්‍යතාවයක් හදාගන්න වෙබ් සයිට් එකක් අත්‍යවශ්‍යයි.

සම්පූර්ණ විස්තරය කියවන්න (Read More)

වෙබ් අඩවියක් හදන්න ප්‍රධාන තාක්ෂණ 3ක් ඕන වෙනවා:

  • HTML (HyperText Markup Language): ගෙයක් හදද්දී බිත්ති බඳිනවා වගේ, වෙබ් සයිට් එකේ ඇටසැකිල්ල (Structure) හදන්නේ HTML වලින්.
  • CSS (Cascading Style Sheets): ගෙදර පාට කරලා, ලස්සන කරලා, සැරසිලි කරනවා වගේ වැඩේ කරන්නේ CSS වලින්. පාට, ෆොන්ට්, හැඩතල ඔක්කොම තීරණය කරන්නේ මේකෙන්.
  • JavaScript (JS): ගෙදරට විදුලිය දීලා, ෆෑන්, ලයිට් වැඩ කරන තත්ත්වයට ගන්නවා වගේ, වෙබ් සයිට් එකට "පණ දෙන්නේ" JS වලින්. උදාහරණයකට බටන් එකක් එබුවම වෙන දේ තීරණය කරන්නේ JS.

Hosting සහ Domain: සයිට් එකක් හැදුවට විතරක් මදි, ඒක ලෝකෙට පෙන්නන්න "Server" එකක දාන්න ඕන (Hosting). ඒ වගේම ඒකට යන්න ලිපිනයක් ඕන (Domain Name - උදා: google.com).

AI (කෘතිම බුද්ධිය) යනු කුමක්ද?

AI හෙවත් Artificial Intelligence කියන්නේ පරිගණකයකට හෝ යන්ත්‍රයකට මිනිසෙකු මෙන් සිතීමට, ඉගෙන ගැනීමට සහ තීරණ ගැනීමට හැකි හැකියාව ලබා දීමයි. මෙය අනාගත ලෝකය සම්පූර්ණයෙන්ම වෙනස් කරන තාක්ෂණයක්. අද වන විට Phone එකේ Face Unlock එකේ සිට ස්වයංක්‍රීයව යන කාර් (Self-driving cars) දක්වා හැමතැනම AI තියෙනවා.

AI වැඩ කරන්නේ කොහොමද?

AI එකක් ඉගෙන ගන්නේ දත්ත (Data) වලින්. අපි දරුවෙක්ට "මේ පූසෙක්, මේ බල්ලෙක්" කියලා පින්තූර පෙන්නලා උගන්වනවා වගේ, පරිගණකයට දත්ත කෝටි ගණනක් දීලා රටාවන් (Patterns) හඳුනාගන්න පුරුදු කරනවා. මේකට Machine Learning කියලා කියනවා.

AI වර්ග:

  • Narrow AI: එක වැඩක් විතරක් කරන්න පුළුවන් AI. උදා: Siri, Google Assistant, ChatGPT. මේවාට හැඟීම් දැනෙන්නේ නෑ.
  • General AI: මිනිහෙක් වගේම ඕනම වැඩක් කරන්න පුළුවන්, තනිවම හිතන්න පුළුවන් AI. මේවා තාම පර්යේෂණ මට්ටමේ තියෙන්නේ.

අනාගතයේදී වෛද්‍ය විද්‍යාව, අධ්‍යාපනය, ප්‍රවාහනය වගේ හැම ක්ෂේත්‍රයක්ම AI නිසා දියුණු වෙනවා වගේම, සමහර රැකියා නැතිවෙන්නත් පුළුවන්.

පරිගණකයක් Slow වෙන්න ප්‍රධානම හේතුවක් තමයි පරණ HDD (Hard Disk Drive) තාක්ෂණය භාවිතා කිරීම. අද කාලේ අලුත් පරිගණක වල එන්නේ SSD (Solid State Drive). මේ දෙකම කරන්නේ දත්ත ගබඩා කිරීම වුනාට, මේ දෙක වැඩ කරන විදිය අහසට පොළව වගේ වෙනස්.

සම්පූර්ණ විස්තරය කියවන්න
  • HDD (Hard Disk Drive): මේක ඇතුලේ තියෙන්නේ වේගයෙන් කැරකෙන චුම්බක තැටි (Magnetic Disks). ඩේටා කියවන්න පොඩි ඉඳිකටුවක් වගේ කොටසක් තැටිය උඩ එහා මෙහා යනවා. මේක යාන්ත්‍රික (Mechanical) නිසා වේගය අඩුයි, සද්දේ එනවා, වැටුනොත් ඉක්මනට කැඩෙනවා.
  • SSD (Solid State Drive): මේකේ කැරකෙන කොටස් මුකුත් නෑ. Pen Drive එකක් වගේ මෙමරි චිප්ස් වලින් හැදිලා තියෙන්නේ. ඒ නිසා HDD එකට වඩා 10 ගුණයක් විතර වේගවත්. පරිගණකය On වෙන්න යන්නේ තත්පර ගානයි.

ඔයාගේ ලැප්ටොප් එක Slow නම්, අලුත් එකක් ගන්න කලින් HDD එක ගලවලා SSD එකක් දාලා බලන්න. ඒක අලුත් මැෂින් එකක් වගේ වැඩ කරයි.

RAM එක කියන්නේ මොකක්ද?

RAM (Random Access Memory) කියන්නේ පරිගණකයේ "කෙටි කාලීන මතකය" (Short-term memory). ඔයා දැනට කරමින් පවතින වැඩ, ඕපන් කරලා තියෙන සොෆ්ට්වෙයාර් තාවකාලිකව ගබඩා වෙලා තියෙන්නේ මේකේ. පරිගණකය Off කරපු ගමන් RAM එකේ තියෙන දේවල් මැකෙනවා.

සම්පූර්ණ විස්තරය කියවන්න

ඇයි RAM එක වැදගත්?

RAM එක වැඩි වෙන තරමට ඔයාට එකපාර වැඩ ගොඩක් කරන්න පුළුවන් (Multitasking). උදාහරණයක් විදියට Chrome එකේ Tabs 10ක්, Photoshop සහ Word එකපාර ඕපන් කරලා තියන්න RAM එක ලොකු වෙන්න ඕන.

  • 4GB: සාමාන්‍ය Office වැඩ සහ ඉන්ටර්නෙට් යන්න ඇති.
  • 8GB: අද කාලේ අවම අවශ්‍යතාවය මෙයයි. Games ගහන්න, පොඩි Design වැඩ වලට හොඳයි.
  • 16GB+: Video Editing, Gaming සහ බර වැඩ වලට අනිවාර්යයි.

RAM එකේ වේගය මනින්නේ MHz වලින්. DDR3, DDR4, DDR5 කියලා පරම්පරා තියෙනවා. අලුත් පරම්පරාව වේගවත්.

Processor (CPU) ගැන දැනගමු

CPU (Central Processing Unit) කියන්නේ පරිගණකයේ "මොළය". අපි පරිගණකයට දෙන හැම විධානයක්ම ක්‍රියාත්මක කරන්නේ, ගණන් හදන්නේ මේකෙන්. මැෂින් එකේ වේගය තීරණය කරන ප්‍රධානම සාධකය මෙයයි. ලෝකයේ ප්‍රධාන CPU වර්ග දෙකක් තියෙනවා: Intel සහ AMD.

සම්පූර්ණ විස්තරය කියවන්න

Processor එකක් ඇතුලේ තව පොඩි Processors ගොඩක් තියෙනවා වගේ හිතන්න. ඒවාට කියන්නේ Cores කියලා (Dual Core, Quad Core). Cores වැඩි වෙන තරමට එකපාර කරන්න පුළුවන් වැඩ ප්‍රමාණය වැඩියි.

  • i5: මධ්‍යම ප්‍රමාණයේ වැඩ, Games සහ සාමාන්‍ය Editing වලට. (බහුලවම පාවිච්චි වෙන්නේ මෙයයි).
  • i7/i9: බරසාර වැඩ, 4K Video Editing, 3D Rendering සඳහා.

Generation (පරම්පරාව) අලුත් වෙන තරමට (උදා: 10th Gen vs 13th Gen) වේගය සහ කාර්යක්ෂමතාවය වැඩියි.

VPN පාවිච්චි කරන්නේ ඇයි?

VPN (Virtual Private Network) මගින් කරන්නේ ඔයාගේ අන්තර්ජාල සම්බන්ධතාවය ආරක්ෂිත උමගක් (Tunnel) හරහා යවන එක. සරලවම කිව්වොත්, ඔයා ඉන්ටර්නෙට් එකේ කරන දේවල් වෙන අයට (Service Provider ට හෝ Hacker ලට) පේන්නේ නෑ. ඔයා ඉන්න රට වෙනස් කරලා පෙන්නන්නත් මේකෙන් පුළුවන්.

VPN පාවිච්චි කරන්න හොඳ වෙලාවල්:

  • Blocked Sites වලට යන්න: සමහර රටවල Block කරපු වෙබ් අඩවි වලට යන්න VPN උදව් වෙනවා.
  • Privacy ආරක්ෂා කරගන්න: ඔයා කරන දේවල් ගැන ඔත්තු බලන එක නවත්තන්න.

Free VPN පාවිච්චි කරද්දී පරිස්සම් වෙන්න, මොකද සමහර ඒවා අපේ Data හොරෙන් ගන්නවා.

Bitcoin සහ Crypto සරලව

Cryptocurrency කියන්නේ ඩිජිටල් මුදල් වර්ගයක්. රුපියල්, ඩොලර් වගේ නෙවෙයි, මේවා අතින් අල්ලන්න බෑ. මේවා පවතින්නේ පරිගණක ජාලයක් තුළ (Blockchain). Bitcoin කියන්නේ ලෝකයේ පළමු සහ ජනප්‍රියම Crypto වර්ගයයි. මේවා පාලනය කරන්න බැංකුවක් හෝ ආණ්ඩුවක් නෑ.

සම්පූර්ණ විස්තරය කියවන්න

Blockchain තාක්ෂණය හරහා තමයි මේ හැම ගනුදෙනුවක්ම සටහන් වෙන්නේ. ඒ නිසා හොර කරන්න අමාරුයි. ලෝකේ ඕනම කෙනෙක්ට තත්පර ගානකින් සල්ලි යවන්න මේකෙන් පුළුවන්.

අවදානම:

Crypto වල වටිනාකම ඉතා ඉක්මනින් ඉහළ පහළ යනවා. අද ලක්ෂයක් වටිනා දේ හෙට දාහක් වෙන්න පුළුවන්. ඒ වගේම මේ ක්ෂේත්‍රයේ වංචාකරුවන් (Scammers) බහුලයි. ඒ නිසා හොඳට ඉගෙන ගෙන මිසක් ආයෝජනය කරන්න එපා.

Android ද? iPhone (iOS) ද?

ස්මාර්ට් ෆෝන් ලෝකයේ තියෙන ලොකුම තරඟය. Google ලගේ Android සහ Apple ලගේ iOS. මේ දෙකේම වාසි අවාසි තියෙනවා. ඔයාට හරියන්නේ මොකක්ද කියලා තීරණය කරන්න මේ කරුණු උදව් වෙයි.

සම්පූර්ණ විස්තරය කියවන්න
  • Android (Samsung, Xiaomi, Pixel): නිදහස වැඩියි. අපිට ඕන විදියට ෆෝන් එක වෙනස් කරගන්න (Customize) පුළුවන්. මිල ගණන් පරාසය පුළුල් (ලාබෙටත් තියෙනවා). Apps හිඟයක් නෑ. හැබැයි සමහර විට ලෙඩ දෙන ගතිය වැඩියි.
  • iOS (iPhone): ඉතාම සරලයි (User Friendly). ආරක්ෂාව (Security) අතින් ඉතා ඉහළයි. අවුරුදු ගානක් ගියත් ෆෝන් එක Slow වෙන්නේ නෑ. Video Quality එක සුපිරි. හැබැයි මිල අධිකයි වගේම නිදහස සීමිතයි.

සරලවම: ඔයාට වැඩ කෑලි දාලා ෆෝන් එක එක්ක සෙල්ලම් කරන්න ඕන නම් Android. ඔයාට කරදරයක් නැතුව වැඩේ කරගන්න සහ හොඳ කැමරාවක් ඕන නම් iPhone.
